資料管理（GAS）

登入後可編輯以下三個資料集：About、Providers、Site Content。

請先在 js/config.js 設定 GAS_BASE_URL，並部署 Apps Script Web App。
SECURITY WARNING

本頁為內部管理界面，未經授權請立即離開。任何嘗試與操作均會被記錄（IP / 時間 / 裝置指紋）。

  • 若你不是系統管理者，請立刻關閉本頁。
  • 持續存取將被視為未經授權行為，系統會自動觸發告警流程。
  • 登入即表示你同意並遵守本平台之資安與資料使用政策。

管理員登入

About（關於頁）

編輯關於頁的標題、導語、模型卡片與成就清單。

四階段模型（model）

成就清單（achievements）

每項可填「文字」即可；若有連結，請一併填入網址。

Site Content（首頁文案）

編輯首頁標題、副標題與平台導覽卡片。

平台導覽卡片（index.platformIntro）

每張卡片可填標題、圖片網址、簡述與補充細節。

Providers 視覺化編輯器（Beta）

此區將資料以表單方式可視化，方便快速增刪改。切換資料集時會自動載入。

來源：目前資料集內容

課程安排（timeline）

每筆包含 time、title、detail。可上下移動或刪除。

精選案例（cases）

- 有圖片（images，多行，每行一個 URL）或影片（video）→ 會顯示為媒體卡片。
- 多筆含媒體 → 卡片網格；僅一筆含媒體 → 橫向交錯。
- 無媒體 → 以文字清單呈現。支援 YouTube/Vimeo/MP4。
